Use this template for full-stack applications that need a clean REST API with Hono on Cloudflare Workers.

Best for:
- REST API backends with a React frontend
- Apps that match VibeSDK's own backend patterns (Hono)
- CRUD applications with D1 database
- Projects where you prefer explicit API routes over RPC

Includes: React 18, TanStack Router (file-based), TanStack Query, Hono for API routes, Cloudflare Workers + D1, 50+ shadcn/ui components, Tailwind CSS with 3 color themes, dark mode support.

Hono provides lightweight, Express-like routing on Cloudflare Workers with middleware support.
